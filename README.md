# 🎭 Felix-Meme-AI

Welcome to **Felix-Meme-AI**, your smart meme generator! This is where Artificial Intelligence meets the ultimate "zueira" (fun). 🚀

## ✨ What does it do?
Felix-Meme-AI transforms any image from your gallery into an epic meme in seconds.
* **Your Face:** Want to see a sarcastic version of a selfie?
* **Screenshots:** Turn that funny chat moment into a joke.
* **Games:** Use an image from your favorite game and let the AI create the caption.

**The Secret:** Since we use **Google Gemini 1.5 Flash** technology, every analysis is unique. You can upload the same photo twice and get **COMPLETELY DIFFERENT** results!

---

## 🚀 How to use it?
1.  Visit the official website: [**Felix-Meme-AI**](https://felixgamesyt.github.io/Felix-Meme-AI/)
2.  Enter your **Gemini API KEY** (the site saves it only in your browser for security).
3.  Choose the meme's vibe: 😂 Funny, 🙄 Sarcastic, 🤓 Nerd, or 😏 Sassy.
4.  Upload your image.
5.  **Download your meme** and share it with everyone!

---

## 🛠️ Built With
* **HTML5 & CSS3:** Modern and responsive layout for mobile.
* **JavaScript:** Processing logic and file handling.
* **Google Gemini AI:** The "brain" that analyzes images and creates captions.
* **html2canvas:** Library used to transform the result into a downloadable image.

---

## 🔒 Security & Privacy
This project was developed with security in mind. Your **API KEY** is NOT stored in the GitHub code (preventing Google from blocking it). It is stored only on your device via `localStorage`.
